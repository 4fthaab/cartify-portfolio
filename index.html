What Cartify Does

Engineered for the
Modern Retail Experience

01
🛒

Smart Cart Interface

Cartify's onboard cart display shows your shopping list in real-time, automatically checking off items as they land in the cart. Built on Raspberry Pi, the UI responds instantly — no phone required during shopping.

Real-time Display Auto Billing QR Login
02
🤖

AI-Powered Item Detection

Items are identified using YOLOv8 computer vision — but detection only triggers when the load cell registers a weight change. This event-driven approach means no continuous camera processing, keeping the system efficient and edge-device friendly.

YOLOv8 Load Cell Sensor Raspberry Pi Frame Differencing Weight Validation
03
🧭

Intelligent Path Navigation

A grid-mapped store layout powers A★ pathfinding, while TSP optimization computes the shortest route through all shopping list items. ArUco marker-based indoor positioning tracks the cart in real time — no GPS needed.

A★ Pathfinding ArUco Markers Grid Mapping Indoor Tracking
04
📱

Mobile App Integration

Shoppers create lists, scan a QR code to pair with their cart, then watch purchases update live. The app handles everything from navigation guidance to digital payment — creating a fully connected shopping loop.

QR Cart Pairing Live Sync Digital Payments UPI / Mock Gateway
05
🏢

Comprehensive Admin Dashboard

Store managers get real-time visibility into every cart, item, and transaction. The dashboard covers inventory management, sales analytics, a visual store layout editor, and instant fraud alerts when weight-vision discrepancies are detected.

Inventory Control Sales Analytics Fraud Alerts Layout Editor
