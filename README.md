# Cartify – Smart Shopping Cart

Cartify is an AI-powered smart shopping cart system that automates billing, optimizes in-store navigation, and enhances the overall shopping experience.

## 🔗 Live Demo

https://cartify.afthabrahman.tech

## ✨ Features

* AI-based item detection (YOLO + weight trigger)
* Automated billing (no barcode scanning)
* Optimized shopping path (A* algorithm)
* Real-time indoor tracking (ArUco markers)
* Mobile app integration (QR cart linking)
* Fraud detection system
* Admin dashboard for store management

## 🛠 Tech Stack

* Frontend: React, Tailwind CSS
* Backend: FastAPI, MongoDB
* AI: YOLOv8n
* Hardware: Raspberry Pi, Load Cell, Camera


A smart retail solution combining AI, IoT, and automation.
